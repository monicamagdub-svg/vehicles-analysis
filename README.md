# Vehicles Analysis Dashboard

Esta aplicación web interactiva permite explorar un conjunto de datos de anuncios de venta de coches en EE.UU. utilizando Streamlit.

## Funcionalidad
- Histograma interactivo del kilometraje (odometer)
- Gráfico de dispersión entre kilometraje (odometer) y precio (price)
- Visualización interactiva con Plotly

## Cómo ejecutar localmente
```bash
streamlit run app.py
